# Revisão Sistemática da Literatura (SLR)
## Disciplina – CESAR School

Este repositório reúne o material utilizado na disciplina de **Revisão Sistemática da Literatura (SLR)** da CESAR School.

O objetivo da disciplina é ensinar, de forma prática, como planejar, conduzir e relatar uma revisão sistemática, mostrando que uma SLR é composta por um conjunto de decisões metodológicas que influenciam diretamente o conhecimento produzido.

Ao longo da disciplina, os alunos percorrem todas as etapas de uma revisão sistemática, desde a definição das questões de pesquisa até a síntese das evidências.

---

# Estrutura da disciplina

A disciplina está organizada em quatro grandes etapas.

```text
Questões de Pesquisa
        │
        ▼
Busca
        │
        ▼
Seleção dos Estudos
        │
        ▼
Extração dos Dados
        │
        ▼
Síntese das Evidências
        │
        ▼
Conhecimento Científico
```

Cada etapa possui objetivos, desafios e técnicas próprias.

# Organização do repositório

## 📂 Extração

A etapa de extração tem como objetivo transformar artigos científicos em dados que possam responder às questões de pesquisa.

Entretanto, **não existe uma única forma de extrair dados**. O que deve ser extraído depende tanto do desenho metodológico dos estudos primários quanto do método de síntese que será utilizado posteriormente.

Nesta pasta são apresentados exemplos de extração para diferentes tipos de estudos.

- Experimentos
- Estudos de Caso
- Surveys
- Relatos de Experiência
- Estudos de Machine Learning

Cada artigo apresenta:

- exemplo fictício;
- formulário de extração;
- comentários sobre quais informações são relevantes;
- gabarito comentado.

---

## 📂 Síntese

Depois que os dados são extraídos, eles precisam ser transformados em conhecimento.

Mas diferentes perguntas de pesquisa exigem diferentes métodos de síntese.

Esta pasta apresenta artigos didáticos sobre os principais métodos utilizados atualmente.

- Análise Bibliométrica
- Síntese Narrativa
- Meta-etnografia
- Metassumário
- Meta-análise
- Síntese Temática
- Realist Synthesis
- Critical Interpretive Synthesis (CIS)

Cada artigo contém:

- explicação do método;
- exemplo completo utilizando um artigo fictício;
- figuras e tabelas ilustrativas;
- guia de referência;
- perguntas para reflexão;
- gabarito comentado.

---

# Como escolher um método de síntese

A escolha do método não deve ser feita porque ele é popular ou frequentemente utilizado na área.

Ela deve ser feita **a partir da pergunta de pesquisa**.

```text
Qual é sua pergunta?

│
├── Quero mapear autores, países, periódicos
│      ↓
│   Análise Bibliométrica
│
├── Quero identificar e organizar estudos
│      ↓
│   Revisão Sistemática
│
├── Quero interpretar os resultados encontrados
│      ↓
│   Síntese Narrativa
│
├── Quero identificar temas recorrentes
│      ↓
│   Síntese Temática
│
├── Quero quantificar a frequência dos achados
│      ↓
│   Metassumário
│
├── Quero medir o tamanho de um efeito
│      ↓
│   Meta-análise
│
├── Quero reinterpretar conceitos
│      ↓
│   Meta-etnografia
│
├── Quero explicar como e por que uma intervenção funciona
│      ↓
│   Realist Synthesis
│
└── Quero questionar pressupostos da literatura
       ↓
   Critical Interpretive Synthesis
```

---

# O que muda entre os métodos?

O mesmo artigo pode gerar tipos completamente diferentes de dados.

| Método | O que é extraído? |
|---------|-------------------|
| Bibliometria | Metadados (autores, países, periódicos, citações...) |
| Revisão Sistemática | Características dos estudos |
| Síntese Narrativa | Resultados e conclusões |
| Síntese Temática | Trechos para codificação |
| Metassumário | Achados |
| Meta-análise | Estatísticas e tamanhos de efeito |
| Meta-etnografia | Conceitos |
| Realist Synthesis | Configurações CMO |
| Critical Interpretive Synthesis | Pressupostos teóricos |

Essa é uma das ideias centrais da disciplina:

> **A estratégia de extração depende diretamente do método de síntese que será utilizado.**

---

# Os métodos de síntese como níveis de abstração

Os métodos desta coletânea diferem não apenas pelas técnicas empregadas, mas também pelo tipo de conhecimento que produzem.

```text
            Maior abstração e construção teórica
                           ↑
        Critical Interpretive Synthesis
                           ↑
              Realist Synthesis
                           ↑
               Meta-etnografia
                           ↑
               Síntese Temática
                           ↑
               Síntese Narrativa
                           ↑
                 Metassumário
                           ↑
                 Meta-análise
                           ↑
      Revisão Sistemática da Literatura
                           ↑
            Análise Bibliométrica
                           ↓
      Maior descrição da literatura
```

Nenhum método é superior aos demais.

Cada um responde a um tipo diferente de pergunta científica.

# Objetivo da disciplina

Ao final da disciplina, espera-se que o estudante seja capaz de:

- elaborar um protocolo de revisão sistemática;
- construir estratégias de busca;
- selecionar estudos de forma transparente;
- definir estratégias de extração consistentes;
- escolher o método de síntese adequado para sua pergunta de pesquisa;
- interpretar criticamente os resultados produzidos por diferentes métodos de síntese;
- relatar revisões sistemáticas seguindo boas práticas metodológicas.

Mais do que aprender técnicas, a disciplina busca desenvolver a capacidade de escolher conscientemente **qual tipo de conhecimento científico se deseja produzir** e quais métodos são mais adequados para esse objetivo.
